# Login Authentication App

This is a simple full-stack Login Authentication project built using
React for the frontend and Node.js with Express for the backend. The
application allows users to log in and access a welcome page after
successful authentication.

This project helps beginners understand how login systems work between a
frontend and backend.

## Features

-   Login form
-   API based authentication
-   Welcome page after login
-   Simple and clean UI

## Technologies Used

Frontend: - React - HTML - CSS

Backend: - Node.js - Express.js

## How to Run the Project

1.  Clone the repository

2.  Install backend dependencies: npm install

3.  Start the backend server: npm start

4.  Install frontend dependencies: npm install

5.  Start the frontend: npm start

6.  Open the browser and visit: http://localhost:3000
